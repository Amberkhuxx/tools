Known Weaknesses

This page lists some known attacks resulting in security of systems running the Jena GRDDL Reader being compromised.

Other known security issues are listed in the conformance section.

The penultimate column shows which versions of the Jena GRDDL Reader are vulnerable. The last column shows information on how to address this vulnerability.

ID Title Description Ver. Sol.
1 Java System Properties

Malicious code can access the Java system properties and communicate them over the web. This is illustrated by the security6 test, which leaks the filepath for the user's home directory.
